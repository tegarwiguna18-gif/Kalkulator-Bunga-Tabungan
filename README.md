# Kalkulator-Bunga-Tabungan
✨ Tentang Aplikasi Kalkulator Bunga Tabungan Bank adalah alat bantu perencanaan keuangan yg akurat,mudah dipahami,&amp; dirancang secara profesional u/ membantumu memperkirakan berapa keuntungan yg akan didapat dari menyimpan uang di bank. Menggunakan rumus bunga sederhana yg berlaku umum,cocok u/ simulasi tabungan harian,bulanan,maupun tahunan
